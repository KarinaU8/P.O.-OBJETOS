El objetivo de simular un sistema de reserva y gestión de vuelos. El proyecto implementa diversas funcionalidades, incluyendo:

- Compra de boletos de avión.
- Reasignación de boletos y asientos.
- Cancelación de boletos.
- Check-in en línea.
- Adición de servicios adicionales durante la reserva.
- Canjeo de millas por descuentos.

ejecutable java -jar Proyecto-POO.jar
